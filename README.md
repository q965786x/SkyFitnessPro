# SkyFitnessPro

Онлайн-тренировки для занятий дома. Достигайте своих фитнес-целей с профессиональными программами.

## О проекте

SkyFitnessPro - это веб-приложение для онлайн-тренировок, которое позволяет пользователям:

- Просматривать доступные курсы по фитнесу, йоге, стретчингу и другим направлениям
- Добавлять курсы в свой профиль
- Отслеживать прогресс выполнения тренировок
- Сохранять результаты упражнений

## Технологии

- **Next.js 14** - React фреймворк
- **TypeScript** - типобезопасный JavaScript
- **Redux Toolkit** - управление состоянием
- **CSS Modules** - стилизация компонентов
- **Axios** - HTTP клиент

## Установка и запуск

### Требования

- Node.js 18+
- npm или yarn

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

### Сборка для production

```bash
npm run build
npm start
```

## API

Приложение использует API:

https://wedev-api.sky.pro/api/fitness

Основные эндпоинты:

POST /auth/register - регистрация

POST /auth/login - авторизация

GET /users/me - данные пользователя

GET /courses - список курсов
